GIT y GITHUB

Objetivo general

Explicar y comprender de que se tratan las herramientas GIT Y GITHUB, desarrollando ejercicios a traves de visual studio code y luego sincronizando lo realizado con GITHUB Desktop para asi que suban y guarden en la nube, para luego poder publicarlos en la web.

Objetivos especificos

  • Identificar los objetivos del proyecto a traves de la herramienta Visual studio code.
  • Construir el codigo del tema escogido, para dar a conocer loq ue se quiere obtener.
  • Crear la pagina web con contenido multimedia, imagenes, texto para que sea entendible el tema de manera eficiente.
  • Desarrollar un vinculo donde se muestre la pagina solicitada demostrando los conceptos aprendidos del tema.
